# HEMS

**HEMS** (Home Energy Management System) is a custom integration for Home Assistant.

## Features

- Local EcoTracker REST API
- Fast 2-second polling
- Grid Power
- Grid Import
- Grid Export
- Modern Home Assistant architecture
- HACS compatible

## Requirements

- Home Assistant 2025.6 or newer
- EcoTracker with REST API enabled

## Installation

### HACS

1. Add this repository as a custom repository.
2. Install **HEMS**.
3. Restart Home Assistant.
4. Add the integration.
